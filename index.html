Site de Vendas

Site feito para trazer credibilidade a empresa e seus produtos, tanto como ser um local para analise, compra e captura de clientes.

Detalhes Site Vendas

O app é renderizado em somente uma pagina, mas uma componentes do bootstrap para simular uma troca de paginas ao clicar na navegação

É utilizado um banco de dados integrado com firebase para atualizar informações da empresa em tempo real, conectando a empresa e seus dados com o site automaticamente

É possivel trocar o thema do site para escuro com um clique e o mesmo fica salvo na sessão do usuario

Existe sistema de carrinho sem login possibilitando que o usuario deixe produtos salvos e faça calculos de quantidade e preço de suas escolhas

Com diversas paginas é possivel organizar o site para promover propagandas, ofertas, produtos que tem mais alcance visual do usuario

O carrossel também é uma forma de divulgar promoções e produtos

Com o filtro de produtos o usuario pode encontrar o produto perfeito para sua demanda

Com o botão detalhes presente em produtos é possivel visualizar mais informações do produto desejado sem poluir a tela
